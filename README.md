## 🛠️ Installation
### With PyPI
```pip3 install cqfd```
### With Github
```bash
git clone https://github.com/megadose/cqfd.git
cd cqfd/
python3 setup.py install
```
## 📈 Usage
```python
from cqfd import *
print(cqfd("Emmanuel Macron"))
```
## 📚 Example with Cqfd.py
```bash
python3 Cqfd.py -n "Emmanuel Macron" -o emmanuel.csv
```
